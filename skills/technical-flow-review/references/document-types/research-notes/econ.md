# Research Notes: Economics Deviations

- Relative to the general research-notes baseline:
  - the note should still reveal what economic question or local issue it concerns
  - even a very internal note should say whether it is about the economic mechanism, identification strategy, theorem statement, estimation issue, or interpretation of results
  - if the note is for a coauthor or advisor, it should mark clearly whether it is reporting progress, proposing a direction, or asking for a concrete decision
- Common additional failure mode:
  - the economics disappears behind local technical detail
  - the note presumes too much shared memory about where the issue sits in the larger paper
