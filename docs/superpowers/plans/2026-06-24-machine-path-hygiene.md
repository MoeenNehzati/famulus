# Machine Path Hygiene — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Eliminate generated runner scripts from the `recurring-tasks` skill directory, move machine-specific addresses to `environment.d` (written at install time), and add a pre-commit hook that prevents machine paths from entering committed source in the future.

**Architecture:** `sync-units.py` stops generating runner scripts and instead writes service files that invoke `run-skill.sh` directly via `/bin/sh -c '"$AI_RUN_SKILL" <job>'`. The installer detects DBUS socket, bin dir, and `run-skill.sh` path at install time and writes them to `~/.config/environment.d/20-ai-agent.conf`. A new pre-commit hook `check-no-machine-paths` scans committed source for `/home/` and `/run/user/<digits>/` patterns and rejects them.

**Tech Stack:** Python 3, bash, systemd user units

```bash
#!/usr/bin/env bash
# run-skill.sh <skill-name>
# Invoked by systemd service ExecStart. Sets up PATH and DBUS, then runs
# the AI agent with the given skill name.
# Derives SKILL_DIR from its own resolved path — no hardcoded paths.
# Logs go to ~/.local/share/ai-recurring/logs/<skill-name>/run.log
# so they live outside the source tree.
set -euo pipefail

SKILL_DIR="$(cd "$(dirname "$(readlink -f "${BASH_SOURCE[0]}")")/.." && pwd)"

# Ensure ~/.local/bin is on PATH (may not be set in systemd user environment)
export PATH="$HOME/.local/bin:$PATH"

# Set DBUS if not already provided (e.g. via environment.d)
if [ -z "${DBUS_SESSION_BUS_ADDRESS:-}" ]; then
    export DBUS_SESSION_BUS_ADDRESS="unix:path=/run/user/$(id -u)/bus"
fi

skill_name="${1:?Usage: run-skill.sh <skill-name>}"

# Logs live outside the source tree under XDG data home
log_dir="${XDG_DATA_HOME:-$HOME/.local/share}/ai-recurring/logs/$skill_name"
mkdir -p "$log_dir"

exec "$SKILL_DIR/scripts/invoke-agent.sh" "$skill_name" \
    >> "$log_dir/run.log" 2>&1
```

- [ ] **Step 1: Write the failing test**

Create `tests/test_no_machine_paths.py`:

```python
#!/usr/bin/env python3
"""Ensure committed source files contain no machine-specific absolute paths."""

from __future__ import annotations
import re, sys
from pathlib import Path

REPO_ROOT = Path(__file__).resolve().parents[1]

CHECK_ROOTS = [
    REPO_ROOT / "skills",
    REPO_ROOT / "references",
    REPO_ROOT / "agents",
    REPO_ROOT / "CLAUDE.md",
]

EXCLUDED_PARTS = {"tests", ".git", ".claude-plugin", ".codex-plugin"}
EXCLUDED_PATHS = {
    Path("skills/install-assistant-tools"),  # install tools may reference paths in docs
}

# Matches /home/<anything> or /run/user/<digits>
MACHINE_PATH = re.compile(r"/home/[^/\s\"']|/run/user/\d+")


def iter_files(path: Path):
    if path.is_file():
        yield path
        return
    if not path.exists():
        return
    for child in path.rglob("*"):
        if not child.is_file():
            continue
        rel_parts = child.relative_to(REPO_ROOT).parts
        if any(part in EXCLUDED_PARTS for part in rel_parts):
            continue
        rel_path = child.relative_to(REPO_ROOT)
        if any(rel_path == ex or ex in rel_path.parents for ex in EXCLUDED_PATHS):
            continue
        yield child


def main() -> int:
    errors: list[str] = []
    for root in CHECK_ROOTS:
        for path in iter_files(root):
            try:
                text = path.read_text(encoding="utf-8")
            except (UnicodeDecodeError, IsADirectoryError):
                continue
            for lineno, line in enumerate(text.splitlines(), start=1):
                if MACHINE_PATH.search(line):
                    rel = path.relative_to(REPO_ROOT)
                    errors.append(f"{rel}:{lineno}: {line.strip()}")
    if errors:
        print("Machine-specific paths found in committed source:")
        for e in errors:
            print(f"- {e}")
        return 1
    return 0


if __name__ == "__main__":
    sys.exit(main())
```
